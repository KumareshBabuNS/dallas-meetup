Solace Messaging on Cloud Foundry


Mahesh Kaleru
Sales Engineer at Solace


This month our presenter will be Mahesh Kaleru from Solace, a strategic partner of Pivotal and the leader in open data movement.

First he will describe the advantages of enabling communications between microservices using open APIs and protocols like AMQP, JMS, MQTT, Qpid, Paho, REST and WebSockets. He will demonstrate how easily you can enable real-time communications between microservices with the new Solace Messaging Tile for Pivotal Cloud Foundry. Then he will discuss the architecture and code that makes it all possible.

Join us to learn how you can improve the way your applications exchange information both within Pivotal Cloud Foundry and across clouds.
